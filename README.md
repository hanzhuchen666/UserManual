## 一、安全注意事项
在未阅读本用户手册之前，请不要拆卸、操作、维护本产品。请不要改装本产品。

#### 1. 危险！！！

- 请确保任何衣物、头发等纤维状物远离工作输出端
- 请不要将本产品输出端对准人或动物
- 未成年人需在成年人监护下使用，15岁以下未成年人禁止使用
- 在手电钻堵转、发出明显异响时立刻停止工作
- 禁止将电池短路、浸水等

#### 2.警告

- 在使用手电钻钻孔、螺丝上紧等功能时，必须固定被操作对象
- 遵循保养规范，按时保养手电钻
- 在手电钻开机之前，确保钻头、螺丝刀头等输出工具被正确夹紧

## 二、主要功能
#### 1. 螺丝、螺母的上紧或旋松

- 通过更换不同规格、形状的螺丝刀头，可以适配不同的螺丝螺母
- 通过旋转扭矩调节圈，可以调整最大输出扭矩，进一步适配不同场合
- 通过拨动速度换挡装置，调节主轴旋转速度为快速档或慢速档
- 通过拨动换向按钮，可以调整输出轴正转、反转，可分别用于螺丝上紧或旋松

#### 2. 在特定材料上钻孔

- 通过更换不同的钻头，可以在不同的材料上钻孔
- 通过调整转速圈、扭矩圈，调整钻孔的输出扭矩、输出转速等，适用于不同场合的钻孔要求

## 四、开箱、安装及检查
用户打开外包装后，请先做下述工作：
1. 参照部件清单，检查下述物品是否齐全：

   | 部件名称               | 个数 |
   | ---------------------- | ---- |
   | 手电钻主体             | 1    |
   | 电池                   | 1    |
   | 电池充电器             | 1    |
   | 配件袋（内含配件若干） | 1    |

2. 安装电池，确保安全的情况下轻叩扳机。观察LED灯，检查电池是否有电。观察输出轴是否能够正常转动。检查设备整体是否有明显异响或大幅度震动。

3. 轻转扭矩限制器外壳，观察是否能够正常转动，检查是否有明显异响。

2. 分别正向、反向转动夹具外壳，观察夹爪是否能够正常伸长和收缩。

#### 2. 需求分析 $$ \longrightarrow $$ 设计理念

| 工作环境/用户需求                                            | 设计理念                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 在使用中，输出轴需要承担较大轴向力；螺丝上紧时需要非常高的同心度。 | 1. 令所有转动轴轴线共线 2. 采用输入输出轴共线的行星减速箱设计 $\longrightarrow$避免了机械振动、减少空间 |
| 用户需要方便的抓手和开关                                     | 外壳呈手枪式设计，方便使用者紧握；将扳机设计为开关，方便用户使用。 |
| 螺丝上紧、钻孔两种功能需求不同的转速、扭矩。                 | 设计装置高低速调节环，在功率相同的情况下提供不同最大转矩。   |
| 防止扭矩过大，导致手电钻脱手伤人                             | 增加扭矩设定环，当转矩超过限制时，第三级行星减速齿圈与减速箱外壳脱离接触 |
| 行星减速箱内，齿圈与齿轮之间需要润滑来保证运行平稳、同时防止磨损 | 减速箱采用闭式设计，内部脂润滑。齿轮、齿轮架均采用淬火处理。 |
| 产品需要能够有较长的使用寿命                                 | TPE（thermoplastic elastomer)外壳，高强度、高回弹，能够长久使用不开裂；易磨损部件采用淬火处理 |

## 七、手电钻的维护
周期性的维护电钻比随机维护电钻更易保持电钻的良好性能。
1. 使用后维护：如果在使用过程中，手电钻收到外界猛烈冲击，则需要检查手电钻的工作状态是否正常。使用后，应将手电钻调至保险挡位、防止误触。使用后应将手电钻放置于干燥、安全、未成年人不能触及的地方储存。
2. 每月维护：检查手电钻的储存环境是否安全
3. 每年维护：检查电池是否有电。将手电钻打开，空转观察是否能够正常工作。以此更换旋转方向，旋转速度，测试手电钻的换向和变速功能是否正常。

## 九、故障分析和排除
| 故障现象| 原因分析 | 故障排除方法 |
|:-----:|:-----:|:-----:|
|工件或夹具松动或旋转|工件或夹具没有完全夹紧在工作台上|使其完全夹紧|
|电源指示灯不亮| 指示灯故障或电池没电 | 检查电源是否有电 |
|工作过程有明显异响| 减速箱齿轮失效或者达到最大扭矩 | 放松扭矩限制，使电钻空载转动。若仍有明显异响，请联系售后服务申请维修 |
